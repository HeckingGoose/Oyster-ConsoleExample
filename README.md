# Oyster - Console Integration

This repository contains an example implementation for Oyster with the .net console application framework.

## Testing it out

To test out the implementation, clone the repository and open the `.sln` file in Visual Studio. Then compile and run the program.

## What's missing?

A lot of game engine related Oyster functionality (sprites, lookers, sound) is missing from this implementation since it is only targetting the console. This implementation is intended to show the bare minimum code required to have Oyster working with some form of display.
